# ChronoPlay - Product Documentation

## Overview
ChronoPlay is a virtual economy and card trading platform that combines elements of collectible card games with social interaction and economic simulation. Players can collect unique cards, trade with friends, manage virtual currency, and participate in a dynamic survival-based economy.

## Core Concept
At its heart, ChronoPlay creates a virtual world where:
- **Cards are valuable assets** with different rarities and properties
- **Cash is the primary currency** for transactions and survival
- **Social connections enable trading** and economic cooperation
- **Time pressure** through survival mechanics keeps the economy active

## User Journey

### 1. **Onboarding**
1. User signs up with email and creates username
2. Email verification ensures account security
3. Initial cash balance provided to start trading
4. Tutorial introduces basic trading concepts

### 2. **Building Network**
1. Discover and add friends
2. Explore available cards
3. Start with simple cash transfers
4. Build trust through small trades

### 3. **Active Trading**
1. Acquire cards through trades or platform additions
2. Propose and execute complex exchanges
3. Manage cash flow and survival tax requirements
4. Build valuable card collections

### 4. **Advanced Participation**
1. Become a key trader in the network
2. Help friends avoid deactivation through strategic trades
3. Optimize portfolio for long-term survival
4. Participate in community building
